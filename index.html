Production

Created using Swift and XCode 6.1, the purpose of this game is to make practicing math skills fun! In Sleepy Sandman, the player plays as the Sandman who is on a quest to fall asleep by counting sheep. However, the game is about more than just counting sheep, but about solving equations and catching the sheep with the correct answer. But avoid the rams! They'll try to bump you out of the sky.

Features

Adorable characters featured in an imaginative nighttime, the game also provides various game modes. The zen mode has no win or lose condition and provides endless math-solving fun. The normal mode lets the player win when 30 sheep are collected.

Release Date

Currently, this game is in development. The target release date is Summer 2015.
